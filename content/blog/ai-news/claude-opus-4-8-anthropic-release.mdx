2026년 5월 28일, Anthropic이 **Claude Opus 4.8**을 공개했어요. 이전 버전인 Opus 4.7이 나온 지 채 두 달도 안 됐는데 업데이트가 또 나왔어요. AI 업계의 릴리즈 속도가 얼마나 빠른지 실감하게 되는 순간이었어요.

## 뭐가 달라졌나요?

Anthropic이 Opus 4.8을 소개하면서 강조한 키워드는 딱 세 가지예요: **더 솔직한 판단**, **더 오랜 자율 작업**, **더 빠른 속도**.

성능 수치를 보면:
- 에이전틱 코딩 점수: 64.3% → **69.2%**
- 도구 사용 다학제 추론: 54.7% → **57.9%**
- Fast Mode 속도: 기존 대비 **약 2.5배 빠름**

특히 '솔직함'에 대한 이야기가 인상적이었어요. Opus 4.8은 자기가 실수했을 때 그걸 인정하는 방식이 더 자연스럽다고 해요. 이전 모델들은 틀린 걸 말해놓고도 어물쩍 넘어가거나 다른 방향으로 얼버무리는 경우가 있었거든요.

## Dynamic Workflows - 더 큰 작업을 더 잘 처리

이번 업데이트에서 가장 흥미로운 기능은 **Dynamic Workflows**예요. 연구 프리뷰 단계인데, Claude Code에서 수백 개의 병렬 서브에이전트를 관리하며 대규모 코드베이스 마이그레이션 같은 작업을 처리할 수 있어요.

쉽게 말하면, 수십만 줄짜리 코드를 리팩토링하는 작업도 Claude가 알아서 쪼개고, 여러 에이전트에 배분하고, 결과를 합쳐서 완성한다는 거예요. 대규모 프로젝트를 혼자 다루는 개발자한테는 꽤 매력적인 기능이에요.

## Effort Control - 얼마나 깊이 생각할지 선택 가능

**Effort Control** 기능도 새로 생겼어요. 사용자가 "extra(xhigh)"나 "max" 모드를 선택하면 모델이 더 많은 토큰을 써서 더 깊이 생각하는 방식이에요.

간단한 질문엔 빠르게 답하고, 복잡한 문제엔 더 많이 고민하도록 직접 설정할 수 있는 거죠. 비용이 늘어날 수 있지만, 정밀도가 중요한 작업엔 좋은 옵션이에요.

## Mythos 모델 예고

Anthropic은 Opus 4.8 발표와 함께 다음 세대 모델인 **Mythos**가 곧 출시된다고 예고했어요. 구체적인 스펙은 공개 안 됐지만, Anthropic의 기업 가치가 9,650억 달러(약 1,300조 원)에 달하는 시점에 내놓는 모델인 만큼 기대감이 커요.

Claude를 자주 쓰는 입장에서, Opus 4.8의 솔직함이나 속도 개선은 체감할 수 있는 수준이에요. AI가 틀렸을 때 당당하게 인정하는 게 사소해 보여도 실제 업무에서는 꽤 중요하거든요.
