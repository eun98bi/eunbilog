2026년 5월 29일, Anthropic이 흥미로운 발표를 했어요. Claude Opus 4.8 출시와 함께 **65억 달러(약 9조 원) 추가 투자**를 유치하면서, 기업 가치가 **9,650억 달러(약 1,300조 원)**에 달했다고 해요. 그리고 그 자리에서 차세대 모델 **Mythos**를 "곧 출시한다"고 예고했어요.

## Mythos, 뭔지 알려진 게 있나요?

솔직히 아직 많지 않아요. Anthropic이 공개한 건 이름과 "몇 주 안에 전체 고객 대상 출시"라는 것뿐이에요. 그래도 몇 가지 단서가 있어요.

**이름의 의미**: Mythos는 그리스어로 '신화', '이야기'를 뜻해요. Claude 모델 계열과는 다른 이름인데, 이게 Claude 시리즈 내 상위 버전인지, 아니면 완전히 새로운 모델 라인인지는 불분명해요.

**공개 채널이 확대됨**: Anthropic이 Mythos의 공개 접근 범위를 확대한다고 밝힌 걸 보면, 이미 제한적으로 테스트 중이었던 것 같아요. 이전에 일부 기업 파트너나 연구자들에게 먼저 공개됐을 가능성이 있어요.

## Opus 4.8 이후 Mythos의 위치

현재 Anthropic 모델 라인업은 이렇게 구성되어 있어요:

| 모델 | 특징 |
|------|------|
| Claude Haiku 4.5 | 빠르고 가벼운 작업용 |
| Claude Sonnet 4.6 | 범용 균형형 |
| Claude Opus 4.8 | 현재 최고 성능 플래그십 |
| **Mythos** | 예정, 더 높은 위치? |

Mythos가 Opus 4.8을 능가하는 완전히 새로운 수준의 모델인지, 아니면 특정 분야에 특화된 모델인지는 아직 모르는 거예요.

## 왜 지금 이 발표를?

1,300조 원의 기업가치를 기록한 시점에 차세대 모델을 예고한 건 전략적으로 이해가 가요. OpenAI의 GPT-5.5와 구글의 Gemini 3.5가 연이어 출시되는 상황에서, "우리도 더 큰 것 가져온다"는 메시지를 던진 거예요.

Anthropic은 줄곧 "안전하고 신뢰할 수 있는 AI"를 핵심 가치로 내세워왔어요. Mythos도 그 방향성 아래에 있을 텐데, 성능과 안전성을 동시에 끌어올릴 수 있을지가 관건이에요.

## 개인적으로 기대되는 것

저는 Claude를 코딩 작업에 꽤 많이 써요. Opus 4.8에서 에이전틱 코딩 점수가 크게 올라서 체감이 됐거든요. Mythos가 나오면 복잡한 코드베이스 작업이나 멀티스텝 작업에서 얼마나 달라지는지 가장 궁금해요.

"몇 주 안에" 출시된다고 했으니, 6월 중에는 소식이 올 것 같아요. 나오면 바로 써보고 후기 올릴게요.
